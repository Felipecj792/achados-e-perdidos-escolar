## 📋 Sobre o Projeto

O **Sistema Achados & Perdidos** é uma aplicação web desenvolvida para a **Escola Estadual Armando Barbosa Quitiba** com o objetivo de centralizar e facilitar o registro, consulta e devolução de objetos perdidos dentro do ambiente escolar.

O sistema elimina a dependência de murais físicos e avisos informais, oferecendo uma plataforma digital intuitiva e acessível para toda a comunidade escolar.

### 🎯 Objetivos

- ✅ Permitir o cadastro de objetos encontrados por alunos, professores e funcionários
- ✅ Possibilitar a busca por objetos perdidos com filtros (categoria, data, local)
- ✅ Notificar o dono quando um item compatível for cadastrado
- ✅ Registrar a devolução do objeto, finalizando o chamado
- ✅ Gerar relatórios simples sobre objetos encontrados, devolvidos e não reclamados
- ✅ Oferecer acesso rápido via dispositivos móveis (design responsivo)

## ✨ Funcionalidades

### Para todos os usuários
- 🔍 **Buscar objetos perdidos** - Pesquise por categoria, local ou palavra-chave
- 👀 **Visualizar detalhes** - Veja informações completas dos objetos encontrados

### Para usuários logados
- 📝 **Cadastrar objetos encontrados** - Adicione itens com foto, descrição e local
- 📋 **Gerenciar meus itens** - Edite, exclua ou marque como devolvido
- 🏷️ **Reivindicar objetos** - Informe quando um objeto é seu

### Para administradores
- 📊 **Painel administrativo** - Visualize estatísticas e relatórios
- 🔄 **Gerenciar devoluções** - Controle centralizado de todos os objetos
- 👥 **Gerenciar usuários** - Administre cadastros da comunidade escolar
